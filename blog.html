Blog Questions

  • Discuss the scope of var, let, and const?

    • Var: var declarations are globally scoped or function scoped. It can be updated. It also a re-declared variables. var can be declared without being initialized.


    let: let declaration are block scoped. It can be updated but not re-declared. let can be declared without being initialized.


    const: const declaration are block scoped. It can not be updated and re-declared. const must be initialized during declaration.


  • Tell us the use cases of null and undefined?

    • null: null means an empty value or a blank value. The data type of null is an object. In arithmetic operations null will be converted to 0 behind the screens. In conditional logic null will return false


    undefined: undefined means the variable has been declared, but its value has not been assigned.The data type of undefined is undefined. In conditional logic undefined will return false. undefined is not an assignment value, as there is no value assigned to the variable, it becomes undefined.

This information, or representation, is delivered in one of several formats via HTTP: JSON, HTML, XLT, Python, PHP, or plain text.
